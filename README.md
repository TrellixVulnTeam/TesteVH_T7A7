 <h2>Como instalar</h2>

Baixe ou clone este repositório.

Dentro do diretório, instale as dependências usando <b>npm install</b>.

 <h2>Como executar</h2>

Execute <b>ng serve</b> para executar a versão de desenvolvimento.

Depois acesse http://localhost:4200/.
